Matlab code and test results for the paper: Mingyuan Zhou, Haojun Chen, John Paisley, Lu Ren, Lingbo Li, Zhengming Xing, David Dunson, Guillermo Sapiro, and Lawrence Carin, "Nonparametric Bayesian Dictionary Learning for Analysis of Noisy and Incomplete Images," submitted to IEEE Trans. Image Process.  PDF

2  Nonparametric

No matter the missing data ratio are 95%, 80%, 20% or 5%, no matter the noise standard deviation are 100, 75, 25 or 5, no matter the images are the gray-scale Barbara, Lena, Peppers or House, the RGB Castle, Mushroom, Kangaroo or Horses, or the hyperspectral Urban or Hill, the code listed above, with exactly the same parameters, can be used to produce state-of-the-art results for all cases.
The results reported in each table below are obtained with the same initialization and parameter setting, the detail can be found in the code listed above:
  • Results in Table 1 can be reproduced by Demo_Denoise_Gray.m
  • Results in Table 2 can be reproduced by Demo_Denoise_RGB.m
  • Results in Table 3 can be reproduced by Demo_Inpainting_GRAY.m
  • Results in Table 4 can be reproduced by Demo_Inpainting_RGB.m
  • Results in Table 5 can be reproduced by Demo_Denoise_Inpainting_GRAY.m
  • Results in Table 6 can be reproduced by Demo_Inpainting_HyperSpecImage.m
